


Написати консольну гру камінь ножиці папір

1. користувач повинен вибирати кількість ігор та ввести своє ім'я
2. користувач повинен мати можливість перервати гру
3. після припинення гри користувач повинен побачити результат: зіграно ігор, виграно, програно
4. Функціонал гри має бути покритий тестами – треба написати тест на метод який визначатиме переможця
5 .Гру упаковати в .jar file і додати його до ДЗ в LMS + лінка на вихідний код
